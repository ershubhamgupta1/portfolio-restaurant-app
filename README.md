## ðŸ›  Installation  

1. **Clone the repository**  
   ```bash  
   git clone https://github.com/ershubhamgupta1/portfolio-restaurant-app  
   cd portfolio-restaurant-app  
   ```  

2. **Install dependencies**  
   ```bash  
   npm install  
   ```  

3. **Run the development server**  
   ```bash  
   npm run dev  
   ```  

4. **Build for production**  
   ```bash  
   npm run build  
   npm start  
   ```
